# Task 4: Complex Form Validation and Dynamic DOM Manipulation

This project demonstrates:

- Stronger client-side form validation including password strength rules
- Dynamic DOM updates for live preview, counters, and progress tracking
- Hash-based client-side routing for a smoother multi-step flow
- Express and EJS for serving the application portal

## Run

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000)

## Test

```bash
npm test
```
