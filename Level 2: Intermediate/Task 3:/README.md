# Task 3: Advanced CSS Styling and Responsive Design

This project demonstrates a more polished responsive webpage using:

- A multi-section landing page layout
- Advanced CSS styling with transitions, gradients, shadows, and animation
- Bootstrap 5 for a responsive and consistent UI system
- Express and EJS for server-side rendering

## Run

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000)

## Test

```bash
npm test
```
